# EStore
EStore is an E-Commerce Web Application built using Microservice Architecture. The backend is built with .Net Core Web APIs and the frontend using React.

#### Progress
| Components         | Status        |
| ------------------ |:-------------:|
| Product Service    | Complete      |
| Review Service     | planning      |
| Search Service     | yet to start  |
| User Service       | yet to start  |
| Auth Service       | yet to start  |
| Cart Service       | yet to start  |
| React Frontend     | UI mocks ready|
